# Seacrets
A sea where your secrets are anonymous and globally available to be ready by anyone sealed in a bottle...

# Why build ts?
I wanted to build something fun, so i created Seacrets for everyone to share their secrets anonymously without being tracked or asked any personal details.. They can share what they want to...

# Tech stack
- Next TS
- Tailwind CSS
- Howler library to fast load sound effects
- Framer motion for animations
- Redis to store Seacrets for faster read n write
- Google fonts

# Installation
First create a `.env` file with `REDIS_URL` env variable.
1. Clone the repo
2. `npm run dev`
Done!! visit `localhost:3000` to view the site
